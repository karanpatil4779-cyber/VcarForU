# vCarforU - Premium Car & Bike Rental Platform

A modern, highly engaging car and bike rental marketplace built with Vite, React, and Tailwind CSS.

## 🚀 Features

- **Interactive Map**: Real-time Leaflet integration to find nearby vehicles and agencies.
- **Rich Search**: Advanced filtering by category, price, fuel type, and transmission.
- **Agency Dashboard**: Comprehensive management tool for agencies to track fleet, earnings, and bookings.
- **Modern Auth**: Unified login for both Customers and Agencies with a polished UI.
- **Glassmorphism UI**: Beautifully designed components with smooth Framer Motion animations.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.

## 🛠️ Tech Stack

- **Frontend**: Vite + React (TypeScript)
- **Styling**: Tailwind CSS + Framer Motion
- **Icons**: Lucide React
- **Maps**: Leaflet + React Leaflet
- **Routing**: React Router DOM
- **Deployment**: Optimized for Vercel

## 🏁 Getting Started

1. **Clone & Install**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## 🌍 Deployment

### Deploying on Vercel
1. Push your code to a GitHub repository.
2. Go to [Vercel](https://vercel.com) and click "Add New Project".
3. Select your repository.
4. Vercel will automatically detect Vite and apply the build settings.
5. Click **Deploy**.

The included `vercel.json` ensures that SPA routing works perfectly on Vercel.

## 📝 Mock Data
The project currently uses high-quality mock data for vehicles and agencies. You can connect it to Firebase or a custom backend by updating the hooks in `src/hooks/`.
